GUÍA DE IMÁGENES – Vive Bien Inmobiliaria
==========================================

Reemplaza cada archivo con la imagen real correspondiente.
Formatos recomendados: .jpg o .webp para mejor rendimiento.

IMÁGENES REQUERIDAS:
--------------------

hero-bg.jpg
  Tipo: Foto principal del hero
  Descripción: Fachada moderna de edificio residencial premium, o vista panorámica
               de Lima desde un rooftop / piso alto. Horizontal, alta resolución.
  Tamaño recomendado: 1920x1080px mínimo

banner-bg.jpg
  Tipo: Banner intermedio CTA (sección Bono Verde)
  Descripción: Interior de departamento modelo amoblado. Sala luminosa, acabados modernos.
  Tamaño recomendado: 1920x900px

proyecto1.jpg
  Tipo: Imagen de tarjeta de proyecto
  Descripción: Render o foto de fachada del proyecto en Miraflores.
  Tamaño recomendado: 800x500px (ratio 16:10)

proyecto2.jpg
  Tipo: Imagen de tarjeta de proyecto
  Descripción: Render nocturno o diurno del edificio en San Isidro.
  Tamaño recomendado: 800x500px (ratio 16:10)

proyecto3.jpg
  Tipo: Imagen de tarjeta de proyecto
  Descripción: Foto de sala o dormitorio modelo amoblado (Surco).
  Tamaño recomendado: 800x500px (ratio 16:10)

mapa-lima.jpg
  Tipo: Sección de Ubicación
  Descripción: Mapa estilizado de Lima con los distritos marcados (Miraflores, San Isidro,
               Surco), o foto aérea de zona residencial. Relación 4:3.
  Tamaño recomendado: 800x600px

cliente1.jpg
  Tipo: Avatar de testimonio
  Descripción: Foto de cliente real (con permiso firmado) o ilustración de persona neutral.
  Tamaño recomendado: 100x100px (cuadrada)

cliente2.jpg
  Tipo: Avatar de testimonio
  Tamaño recomendado: 100x100px (cuadrada)

cliente3.jpg
  Tipo: Avatar de testimonio
  Tamaño recomendado: 100x100px (cuadrada)

NOTAS:
------
- Optimiza todas las imágenes con TinyPNG (https://tinypng.com) o Squoosh antes de subir.
- Considera usar formato .webp para mejorar velocidad de carga (hasta 30% más ligero).
- Para producción, usa un CDN para servir las imágenes.
- Las imágenes de fondo (hero-bg, banner-bg) tienen un overlay oscuro en el CSS,
  por lo que pueden tener colores saturados y aun así verse bien.
